# Mr-clearance
# ModernTech Solutions — HR Management System

## Project Description
A front-end web-based HR management system built for 
ModernTech Solutions as part of the LCA YouthCode 
Course 1 Core Project. The system provides HR staff 
with a centralised platform to manage employee data, 
process payroll, track attendance, and handle leave 
requests.

## How To Run The Project
1. Download or clone the repository
2. Open the project folder in VS Code
3. Right-click on index.html
4. Click "Open with Live Server"
5. The app will open in your browser automatically
6. Login with username: admin and password: admin123

## Technologies Used
- Vue.js 3 (via CDN) — JavaScript framework for 
  building the user interface
- Bootstrap 5 (via CDN) — CSS framework for 
  responsive design and styling
- HTML5 — Page structure
- CSS3 — Custom styling
- JavaScript ES6 — Application logic
- localStorage — Client-side data persistence
- Git and GitHub — Version control and hosting

## Key Features
- Mock authentication system with login and logout
- Employee management — view, search, filter and 
  add employees
- Payroll processing — automated calculations for 
  PAYE, UIF and pension with digital payslip generation
- Leave management — submit, approve and deny 
  leave requests
- Attendance tracking — daily attendance with 
  Present, Late and Absent status
- Dashboard — real time overview of all HR data
- Fully responsive design for mobile and tablet
- localStorage for client-side data persistence

## Business Requirements Addressed
- Centralised employee data management
- Automated payroll calculations
- Digital payslip generation
- Time off request and approval system
- Daily attendance tracking
- Responsive interface for remote HR staff
